Final clauses (i.e., expressing purpose) have their verbs in the subjunctive, introduced by *ut* when positive, by *ne* when negative.
>**Example**<br>
>Laborant pauperes ut divites fiant – poor men work to become rich.<br>
>Se receperunt ne consilia ab hostibus cognoscerentur – they retreated in order that their plans might not be discovered by the enemy.

## Sequence of tenses

| **A. Primary tenses–**  |             |
| ----------------------- | ----------- |
| <small>*apodosis*</small>              | <small>*protasis*</small>  |
| Present.                | Present.    |
| Future.                 | Perfect.    |
| Perfect with 'have.'    |             |
| **B. Historic tenses–** |             |
| Imperfect.              | Imperfect.  |
| Perfect without 'have.' | Pluperfect. |
| Pluperfect              |             |

### Exercise 3A

| English                                                             | Latin                                                                   | Notes                                                                        |
| ------------------------------------------------------------------- | :---------------------------------------------------------------------- | ---------------------------------------------------------------------------- |
| The enemy retreated in order to avoid a battle                      | hostēs se <s>reciperent</s> ut proelium vītārent                        | - retreated is simple historic, so indicative (*rēcepērunt*).                |
| We shall send 200 men in order that we may hinder the enemy's march | mittēmus ducentōs virōs ut iter hostium impediāmus                      |                                                                              |
| They marched quickly so that the enemy might not learn their plans  | celeriter contendērunt nē hostēs <s>consilium nostrum</s>  cognoscārent | - <u>their</u> plans (pl) (*consilia sua*)                                   |
| We advanced to the top of the hill to see the enemy's camp          | ad summum collis prōgrediēbantur ut castra hostium vidērent.            | - *ad* takes acc., so top (adjective) of the hill should be *summum collem*. |
| We work in order to become rich                                     | laborāmus ut divitēs <s>fīant.</s>                                      | - verb protasis is wrong person, should be *fīāmus*.                         |
| They have come to ask for arms from us                              | venērunt ut arma nōbis rogent                                           |                                                                              |
| In order that we may not think you a coward, fight bravely          | <s>validus pugnat</s> nē tē ignāvum esse putēmus.                       | - *fortiter pugna* (imperative).                                             |
